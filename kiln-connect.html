KILN CONNECT

Open source kiln and shop monitoring. Real data, local server.
● ACTIVE — OPEN SOURCE
// ABOUT THIS PROJECT //

Kiln Connect monitors kiln temperature, shop humidity, and ambient conditions, then uploads that data to a local web server so you can check in from anywhere on your network.

Built around accessible hardware — designed to be cheap to build, easy to modify, and completely open source. Files are available for anyone to use or improve.

This is an evolving project. New sensors and features get added as the shop needs change.
